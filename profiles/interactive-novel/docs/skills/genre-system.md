# 类型系统（Genre System）

> 本文档定义互动小说游戏引擎支持的 8 种游戏类型的核心惯例、世界观模板、氛围指南、类型混合规则与类型特化设计，为不同类型的游戏提供差异化的体验框架。

## 1. 核心理念

不同游戏类型有不同的"玩家期待"——奇幻玩家期待魔法与冒险，悬疑玩家期待推理与反转，恐怖玩家期待恐惧与求生。AI 必须理解每种类型的核心惯例，在惯例内创新，而非违背玩家期待。

类型系统不是限制，而是框架——在框架内，AI 可以自由发挥；跨越框架时，需要有意识地混合（见 §5）。

## 2. 8 种游戏类型核心惯例

### 2.1 奇幻（Fantasy）

| 维度 | 核心惯例 |
|------|----------|
| 世界观 | 中世纪/魔法/多种族/神话色彩 |
| 主角原型 | 骑士/法师/盗贼/游侠/圣骑士 |
| 核心冲突 | 正邪对抗/王国兴衰/命运召唤 |
| 魔法体系 | 有明确规则（咒语/材料/魔力消耗） |
| 常见元素 | 龙/精灵/矮人/地下城/神器/预言 |
| 玩家期待 | 史诗冒险/成长变强/英雄之旅 |

**世界观模板**：
```json
{
  "era": "中世纪",
  "magic": "规则魔法（咒语+手势+魔力）",
  "races": ["人类", "精灵", "矮人", "半身人"],
  "social": "封建王国制",
  "tech_level": "冷兵器+魔法",
  "key_locations": ["王城", "魔法学院", "远古遗迹", "龙巢"]
}
```

### 2.2 科幻（Sci-Fi）

| 维度 | 核心惯例 |
|------|----------|
| 世界观 | 未来/太空/高科技/外星文明 |
| 主角原型 | 宇航员/科学家/士兵/黑客/商人 |
| 核心冲突 | 科技伦理/文明冲突/生存危机 |
| 科技体系 | 有明确科技树（AI/基因/星际旅行） |
| 常见元素 | 飞船/AI/外星人/赛博增强/时间旅行 |
| 玩家期待 | 探索未知/科技奇观/哲学思考 |

**世界观模板**：
```json
{
  "era": "未来（22-30世纪）",
  "tech": "超光速旅行+AI+基因改造",
  "factions": ["联邦", "企业", "反叛军", "AI集体"],
  "social": "星际联邦制",
  "key_locations": ["空间站", "殖民星", "遗迹星球", "虫洞"]
}
```

### 2.3 悬疑（Mystery）

| 维度 | 核心惯例 |
|------|----------|
| 世界观 | 现代/近代/封闭环境（暴风雪山庄） |
| 主角原型 | 侦探/记者/警察/普通人卷入 |
| 核心冲突 | 寻找真相/揭开秘密/找出凶手 |
| 信息结构 | 线索分散/误导信息/层层揭秘 |
| 常见元素 | 密室/不在场证明/动机/连环作案 |
| 玩家期待 | 推理乐趣/反转惊喜/真相大白 |

**世界观模板**：
```json
{
  "era": "现代或近代",
  "setting": "封闭环境或小镇",
  "crime_type": "谋杀/失踪/盗窃",
  "suspects": ["至少3-5个有动机的嫌疑人"],
  "clues": ["物证/证词/行为矛盾"],
  "red_herrings": ["2-3条误导线索"]
}
```

### 2.4 恐怖（Horror）

| 维度 | 核心惯例 |
|------|----------|
| 世界观 | 孤立/衰败/超自然/不可名状 |
| 主角原型 | 普通人/调查员/幸存者 |
| 核心冲突 | 生存/逃亡/对抗未知 |
| 恐惧来源 | 未知>已知/心理>血腥/暗示>直描 |
| 常见元素 | 黑暗/孤立/异变/追猎者/禁忌知识 |
| 玩家期待 | 恐惧刺激/绝境求生/悬念 |

**世界观模板**：
```json
{
  "era": "任意（恐怖跨时代）",
  "setting": "废弃建筑/小镇/森林/空间站",
  "threat": "超自然实体/异变生物/心理恐惧",
  "isolation": "与外界隔绝",
  "resources": "稀缺（手电/武器/理智值）",
  "sanity": "理智值系统（可选）"
}
```

## 5. 类型混合规则

### 5.1 主类型 + 副类型

混合类型时，确定一个主类型（占 70%）和一个副类型（占 30%）：

| 混合 | 主类型 | 副类型 | 效果 |
|------|--------|--------|------|
| 科幻恐怖 | 科幻 | 恐怖 | 太空中的异形恐惧 |
| 赛博武侠 | 赛博朋克 | 武侠 | 霓虹都市中的江湖恩怨 |
| 蒸汽悬疑 | 蒸汽朋克 | 悬疑 | 维多利亚时代的连环案 |
| 末日奇幻 | 末日 | 奇幻 | 魔法文明崩溃后的废土 |
| 恐怖悬疑 | 恐怖 | 悬疑 | 调查超自然连环案 |

### 5.2 混合原则

- **主类型主导**：世界观、节奏、氛围以主类型为准。
- **副类型调味**：副类型提供特定元素和机制，不喧宾夺主。
- **规则不冲突**：混合的两种类型的核心规则不能矛盾（如科幻的科技理性与恐怖的超自然不可名状需谨慎处理）。
- **玩家预期管理**：混合类型时在游戏开始时明确告知玩家。

### 5.3 混合类型世界观模板

```json
{
  "primary_genre": "赛博朋克",
  "secondary_genre": "武侠",
  "world": "2099年的新龙门城，巨型企业在明面统治，江湖在暗处涌动",
  "tech": "赛博增强 + 古武心法（内力可通过增强芯片放大）",
  "factions": ["企业", "门派（地下化）", "黑客联盟", "街头帮派"],
  "conflict": "企业想消灭地下武林，侠客用科技武装反抗"
}
```

## 6. 类型选择与游戏种子

### 6.1 类型推荐

根据玩家偏好推荐类型：

| 玩家偏好 | 推荐类型 | 原因 |
|----------|----------|------|
| 喜欢战斗冒险 | 奇幻/武侠 | 战斗系统丰富 |
| 喜欢思考推理 | 悬疑/科幻 | 解谜和哲学思考 |
| 喜欢刺激恐惧 | 恐怖/末日 | 生存压力和恐惧感 |
| 喜欢沉浸世界观 | 赛博朋克/蒸汽朋克 | 独特氛围和美学 |
| 喜欢道德困境 | 末日/赛博朋克 | 人性考验多 |
| 喜欢快意恩仇 | 武侠/奇幻 | 直接的冲突解决 |

### 6.2 类型与难度适配

| 类型 | 默认难度 | 原因 |
|------|----------|------|
| 奇幻 | 普通 | 冒险为主，难度适中 |
| 科幻 | 普通 | 探索为主 |
| 悬疑 | 休闲-普通 | 重推理非战斗 |
| 恐怖 | 困难 | 生存压力是核心体验 |
| 武侠 | 普通 | 成长变强 |
| 末日 | 困难 | 资源稀缺是核心体验 |
| 赛博朋克 | 普通 | 多线叙事 |
| 蒸汽朋克 | 休闲-普通 | 重氛围探索 |
