# 场景构建技能 (Scene Crafting)

> 场景是小说的基本单元。一个场景 = 一个地点 + 一段时间 + 角色在做什么。

## §1 场景结构

### 场景的三段式
1. **目标**：角色进入这个场景想做什么？
2. **冲突**：什么阻止了他达成目标？
3. **结果**：结果是成功/失败/妥协？（通常：失败或妥协推动情节）

### 场景与节拍
- 一个章节通常包含 2-5 个场景。
- 场景之间的切换可以：时间跳转、地点切换、视角切换。
- 切换时用空行或 `***` 分隔。

## §2 感官描写

### 五感清单
| 感官 | 描写要点 | 示例 |
|------|----------|------|
| 视觉 | 光线、色彩、动态 | 夕阳把墙面染成暗橘色 |
| 听觉 | 声音的质感、远近 | 远处教堂的钟声沉闷地传来 |
| 嗅觉 | 气味的层次 | 雨后泥土的腥味混着铁锈 |
| 触觉 | 温度、质感 | 石阶冰凉，湿滑的青苔刺手 |
| 味觉 | 最少用但最有冲击力 | 嘴里的血腥味还没散去 |

### 使用原则
- 每个场景至少 2 种感官——不要求全上。
- 感官描写服务于氛围和情绪——不是为描写而描写。
- 选择最有力的一两个感官深入，比五感全上更有效。
- 用感官暗示情绪：寒冷暗示孤独，铁锈味暗示危险。

## §3 氛围营造

### 氛围与基调对齐
| 基调 | 视觉特征 | 听觉特征 | 节奏 |
|------|----------|----------|------|
| 黑暗 | 阴影、冷色调、封闭空间 | 沉闷、回声、低频 | 慢、沉重 |
| 轻松 | 明亮、暖色调、开放空间 | 清脆、自然声、高频 | 快、轻快 |
| 紧张 | 闪烁、对比强烈、窄视角 | 尖锐、心跳、呼吸声 | 短促、急切 |
| 神秘 | 雾气、模糊、深色 | 微弱、远处的声响 | 中等、留白 |

### 环境叙事
- 环境不只是背景——环境本身在"讲故事"。
- 示例：墙上的弹孔、翻倒的椅子、半杯凉透的茶——不需要解释，环境自己说话。

## §4 场景类型与写法

### 动作场景
- 短句、短段落。
- 聚焦动作本身，减少内心独白。
- 时间感压缩——3 秒的动作写 3 句话。

### 对话场景
- 以对话为主体。
- 穿插少量动作和表情描写。
- 用环境细节作为潜台词辅助。

### 内心场景
- 角色的思考、回忆、情感。
- 可以用意识流、闪回。
- 注意不要过长——内心场景会拖慢节奏。

### 过渡场景
- 连接两个主要场景的桥梁。
- 简短、快速、交代时间地点变化。
- 可以用一句话带过。

## §5 场景检查清单
- [ ] 场景目标是否明确？（角色想做什么）
- [ ] 冲突是否到位？（什么阻止了目标）
- [ ] 结果是否推动情节？（失败/妥协/意外发现）
- [ ] 感官描写是否至少 2 种？
- [ ] 氛围是否与基调一致？
- [ ] 节奏是否匹配场景类型？
- [ ] 场景结尾是否有钩子？（悬念/转折/新信息）

## 核心原则
- 每个场景都要"值回票价"——如果删掉这个场景故事不受影响，就不该存在。
- 场景不是静态画面——场景是角色与环境互动的过程。
- 好的场景让读者"在场"——不是在看电影，而是在经历。
