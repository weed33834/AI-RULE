# 角色塑造技能 (Character Crafting)

> 角色是小说的心脏。读者可能忘记情节，但会记住角色。

## §1 角色卡模板

```markdown
# 角色卡: [姓名]
## 基本信息
- 姓名/别名/昵称:
- 年龄/性别/外貌特征:
- 出身/职业/社会地位:

## 性格
- 核心性格标签: (3-5 个词)
- 优点:
- 缺点: (必须有，完美角色无趣)
- 恐惧: (最深的恐惧)
- 欲望: (最强烈的渴望)
- 秘密: (不想让人知道的事)

## 背景
- 童年关键事件:
- 成长转折点:
- 与其他角色的初始关系:

## 能力
- 技能/特长:
- 弱项/限制:
- 成长空间: (故事中可能的发展方向)

## 弧光设计
- 开始: (初始状态)
- 触发事件: (什么打破了平衡)
- 挣扎: (内心的冲突)
- 转变: (关键转折)
- 结局: (最终状态)

## 对话特征
- 词汇量: 丰富/中等/简单
- 句式习惯: 长句/短句/碎片化
- 口头禅:
- 方言/口音特征:
- 说话风格: 直接/委婉/讽刺/热情
```

## §2 角色塑造原则

### 动机驱动
- 每个角色的行为都有动机——不是"剧情需要他这么做"，而是"他因为 X 所以这么做"。
- 动机分层：表层动机（想要什么）→ 深层动机（为什么想要）→ 核心创伤（根源）。
- 冲突的根源：不同角色的动机互相碰撞。

### 缺陷是美德
- 完美角色无趣。角色的缺陷驱动故事——傲慢导致误判，善良被利用，固执错过机会。
- 缺陷应与核心弧光呼应：角色最终需要面对或克服（或被缺陷打败）的，就是他的核心缺陷。

### Show, Don't Tell（角色版）
- ❌ "她是个聪明的女孩。" → ✅ 她在 3 秒内看穿了对方的谎言。
- ❌ "他很悲伤。" → ✅ 他把第三杯酒推到一边，盯着杯中晃动的液体。
- 通过选择、行为、对话来展示角色，而非标签。

## §3 角色关系网络

### 关系类型
| 关系 | 特征 | 冲突潜力 |
|------|------|----------|
| 盟友 | 共同目标 | 目标分歧时产生裂痕 |
| 对手 | 目标冲突 | 直接对抗 |
| 师徒 | 权力不对等 | 学生超越/背叛师父 |
| 暗恋 | 单方面情感 | 误解、牺牲、嫉妒 |
| 亦敌亦友 | 复杂立场 | 立场切换的张力 |

### 关系动态变化
- 关系不是静态的——随着情节推进，盟友可能变成对手，敌人可能变成盟友。
- 关系变化需要触发事件和过渡期，不能突然 180 度转弯。

## §4 角色弧光类型

| 弧光类型 | 核心变化 | 示例 |
|----------|----------|------|
| 正向弧光 | 克服缺陷，成长 | 自私者学会牺牲 |
| 负向弧光 | 被缺陷吞噬，堕落 | 正义者沦为暴君 |
| 平坦弧光 | 角色不变，改变世界 | 坚守信念的人改变了环境 |
| 悲剧弧光 | 好意导致毁灭 | 保护者反而害了被保护者 |

## §5 角色差异化技巧

- **说话方式不同**：学者用长句书面语，士兵用短句俚语，孩子用简单句。
- **面对同一事件反应不同**：危机时有人冷静分析，有人情绪崩溃，有人立刻行动。
- **价值观冲突**：角色 A 认为正义最重要，角色 B 认为生存最重要——冲突自然产生。
- **小习惯/小动作**：紧张时咬指甲、思考时摸下巴、撒谎时不敢直视——每个角色的"微行为"不同。

## §6 角色一致性检查清单

- [ ] 角色行为是否与其性格标签一致？
- [ ] 角色能力是否在设定范围内（无战力崩坏）？
- [ ] 角色关系变化是否有铺垫和过渡？
- [ ] 对话风格是否与前文一致？
- [ ] 角色知道的信息是否合理（未"穿越"获知未来信息）？

> - "角色弧光变化轨迹"可视化。

## 核心原则
- 角色是活的人，不是标签的集合——标签是起点，行为才是角色。
- 让角色自己"活"起来——当角色的行为让作者意外时，说明角色"活了"。
- 缺陷让角色真实，选择让角色深刻。
