# 对话技巧技能 (Dialogue Crafting)

> 对话是角色的灵魂。好的对话推动情节、揭示角色、制造张力——而不只是"角色在说话"。

## §1 对话的核心功能

每段对话应至少实现以下功能之一：
1. **推动情节**：通过对话传递信息、做出决定、引发冲突。
2. **揭示角色**：通过措辞、语气、内容展示角色性格。
3. **制造张力**：通过潜台词、对立、信息差制造紧张感。
4. **建立关系**：通过对话方式展示角色间的关系亲疏。

- **禁止"废话对话"**：不推动情节、不揭示角色、不制造张力的对话应删除。
- 例外：偶尔的"日常对话"可用于节奏舒缓，但不宜超过 2-3 轮。

## §2 对话个性化

### 词汇层差异
| 角色类型 | 词汇特征 | 示例 |
|----------|----------|------|
| 学者 | 书面语、专业术语 | "这种现象的本质是认知偏差。" |
| 军人 | 短句、命令式、俚语 | "动。现在。别磨蹭。" |
| 孩子 | 简单词、叠词、提问多 | "那个大大的东西是什么呀？" |
| 商人 | 圆滑、模糊、利益导向 | "这事儿嘛，也不是不能谈。" |
| 贵族 | 雅致、含蓄、典故 | "秋雨虽寒，却不及人心凉薄。" |

### 句式层差异
- 长句 vs 短句：紧张时角色说话变短，放松时变长。
- 完整句 vs 碎片句：情绪激动时句子破碎，冷静时完整。
- 疑问 vs 陈述：不自信的角色多问，自信的角色多陈述。

### 口头禅和小习惯
- 每个主要角色可设计 1-2 个语言习惯（不滥用）。
- 示例：总是说"理论上"的学者、总是反问"你觉得呢？"的领导者。

## §3 对话标签

### 基本原则
- "说"是隐形的——读者几乎不会注意到"他说"。
- 尽量少用"喊道""嚷道""低语道"等修饰标签——让对话内容本身传达情绪。
- 用动作代替标签：`"别动。"他把手放在剑柄上。` 比 `"别动。"他警告道。` 更好。

### 标签使用频率
- 连续对话中，一旦确立了说话顺序，可省略标签。
- 3 人以上对话时，需要更频繁的标签或动作来区分说话者。
- 避免每句都有标签——显得机械。

## §4 潜台词

> 角色说的不等于角色想的。好的对话有表面含义和深层含义。

### 潜台词技巧
- **说反话**：角色说"没事"但明显有事。
- **转移话题**：角色回避某个问题，暗示那个问题触到了痛处。
- **过度解释**：角色解释得太多，暗示在掩饰。
- **沉默**：角色不说话——沉默本身是一种回应。

### 示例
```
"你昨晚去哪了？"
"出去走了走。"她整理着桌上的文件，没有抬头。
"走到凌晨三点？"
"嗯。"
他等了十秒。她始终没有抬头。
"好吧。"他转身离开。
```
→ 潜台词：她在回避，他知道她在撒谎，但不想逼她。

## §5 对话节奏

### 快节奏（冲突/紧张）
- 短句、短词。
- 来回快，不让角色"思考太久"。
- 动作穿插少——纯对话交锋。

### 慢节奏（亲密/铺垫）
- 长句、丰富描写。
- 对话间穿插动作和内心。
- 允许角色"跑题"——自然聊天感。

## §6 对话检查清单
- [ ] 每段对话是否至少实现一个核心功能？
- [ ] 不同角色的说话方式是否有区分？
- [ ] 是否有过多的修饰标签（"警告道""怒吼道"）？
- [ ] 潜台词是否到位——角色说的和想的是否有差异？
- [ ] 对话是否推动了情节或揭示了角色？
- [ ] 是否有"废话对话"可以删除？

## 核心原则
- 对话是动作的延伸——角色说话也是在"做"。
- 好的对话像打乒乓球——一来一回有节奏，有力量变化。
- 读者跳过的是描写，但不会跳过好对话。
